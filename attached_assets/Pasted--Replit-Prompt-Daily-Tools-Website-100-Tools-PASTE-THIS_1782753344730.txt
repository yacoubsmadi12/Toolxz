Add AdSense placeholder slots in every page. Use this pattern:

```html

<div class="ad-container ad-leaderboard">
  <ins class="adsbygoogle"
       style="display:block"
       data-ad-client="ca-pub-XXXXXXXXXXXXXXXXX"
       data-ad-slot="XXXXXXXXXX"
       data-ad-format="auto"
       data-full-width-responsive="true"></ins>
  <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
</div>
```

Add the AdSense script in `<head>` of every HTML file:
```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXXX" crossorigin="anonymous"></script>
```

Create 3 ad sizes used throughout:
- `.ad-leaderboard` — 728x90 (top/bottom of pages)
- `.ad-rectangle` — 300x250 (sidebar / in-content)
- `.ad-mobile-banner` — 320x50 (mobile only)

Style ad containers with a subtle `[Advertisement]` label above, gray border, centered.
